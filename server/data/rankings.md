## Method

- Score against the weight definitions in [Real-Estate-Knowledge/01-Buying-Criteria.md](https://github.com/bitflipper1/Real-Estate-Knowledge/blob/8fbc132f201ec5bab37adf2d819b394ca09e7187/01-Buying-Criteria.md).
- Evidence cited in each builder profile under [`builders/`](builders/); full source list in [`sources/source-index.md`](sources/source-index.md).
- This pass distinguishes local-division performance from national reputation where evidence allowed (notably Tri Pointe, Mattamy, Eastwood, Smith Douglas — all have Charlotte/regional-specific data; the rest are national-aggregate only).
- Sample size and date range: all review-platform data gathered 2026-07-04 in a single pass; no historical trend data collected. Complaint counts and star ratings are noted per-builder in each profile rather than repeated here.
- Taylor Morrison is listed last/unranked rather than assumed best — its strong trust-award signal is real but this pass did not locate the buyer-complaint-platform data used to score every other builder, so it isn't comparable on the same basis.
