## Questions, risks, and next actions

- "Warranty is a joke" is a specific, repeated phrase across multiple review sources rather than a generic complaint — treat warranty responsiveness as a known, documented risk for this builder, not a surprise if it materializes.
- The energy-efficiency claims (M.R.E.S., HERS mid-50s–60s) are a genuine potential differentiator for a rental-yield calculation (lower utility costs can support higher achievable rent or better tenant retention) — but they rest on self-reported homeowner data in this pass, not an independent HERS audit. Request the actual HERS certificate for any specific home before relying on this in an investment model.
- Get an independent inspection regardless of the energy-efficiency marketing, given the repeated quality/warranty complaint pattern.
