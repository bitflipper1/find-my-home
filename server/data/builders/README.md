## Note on source-repo access

`bitflipper1/Real-Estate-Knowledge` was private and outside this app's GitHub session scope for most of this research pass; access opened up mid-pass via an anonymous `git clone`, which is read-only. These profiles were authored to match that repo's exact structure and evidence discipline (Snapshot table → Evidence-backed findings with source IDs, claim type, confidence, and as-of dates → Questions/risks → Sources), but the underlying research here is Charlotte-market secondary-source review data (BBB, ComplaintsBoard, ConsumerAffairs, etc.), not the primary-document evidence (contracts, lender worksheets) that repo's standard calls for. Nothing here has been pushed to that repo — this app has no write credentials to it.
